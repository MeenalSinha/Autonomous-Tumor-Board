## 🎯 Key Talking Points (If Asked)

**Technical:**
- Multi-agent architecture (4 specialized agents)
- Disagreement detection between agents
- Confidence scoring per modality
- Guideline-grounded recommendations

**Clinical:**
- No autonomous decisions
- Human review mandatory
- Flags uncertainty explicitly
- Mirrors real tumor board format

**Impact:**
- 99.9% time reduction
- Access for under-resourced hospitals
- Lets doctors focus on decisions, not data gathering

**Safety:**
- Draft preparation tool only
- All reasoning transparent
- Multiple disclaimers
- Escalation logic built-in
